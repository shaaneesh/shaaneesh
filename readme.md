# Standards-Based Living (StaBL)

July 7, 2026 (7-7).
This is the announcement of a new way of thinking about societal and personal values & ideas.  It is intended to apply globally, independent of ideologies, politics, education, doctrine, spiritualities, ethnicities, nationalities, etc.


## Core Concept

If you believe in a value, it should be possible to make it a human standard and apply it equally.  There may be exceptions to the standard, but if so, those exceptions should be clear, documented, specific, and measurable or testable.


## Reference & Ranking

This is defining a standard for priorities or measurement or criticism.
If something is any of the above, it should be possible to set a Reference that is measurable, and then do a Ranking to put them in priority order.
For example, allocating resources to various priorities, placing attention on a concern or criticism, criticizing others' choices or priorities, defining impacts for policies, etc.
As with StaBL, exceptions should be clear and documented.  Some References are not possible to measure numerically, but the concept can still be defined and the person (or people) focused on it should be able to do a Ranking in a neutral way.
